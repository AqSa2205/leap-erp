Expected Excel Format

Your Excel file should have headers in the first row. The following columns are recognized:

Column Name Maps To Required

Title / Project Title Title Optional

Contact Name / Name Contact Name Optional
Contact Email / Email Contact Email Optional
Contact Telephone / Phone Contact Telephone Optional
Contact Address / Address Contact Address Optional
Region Region Optional
Postcode Postcode Optional
